# Change: 添加雅虎 B2B 采购门户 API

## Why
为雅虎 B2B 采购门户实现完整的后端 API 支持，包括用户认证、产品管理和订单处理等核心功能，使雅虎客户能够通过后台系统进行阿里巴巴商品采购。

## What Changes
- 添加基于 JWT 的用户认证系统
- 实现产品列表和详情 API
- 创建订单管理和状态追踪 API
- 添加物流追踪链接功能
- 实现分页和筛选功能
- 添加多币种支持（CNY/JPY）

## Impact
- 受影响的规格: auth, products, orders
- 受影响的代码: 新增 API 控制器、模型、数据库迁移文件
- 新增外部依赖: JWT 认证包、阿里巴巴 API 集成
